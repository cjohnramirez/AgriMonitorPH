# Support Plan

## Support Process
Users can report issues via the GitHub Issues tab or by emailing `support@agrimonitor.ph`.

## Common Issues & Troubleshooting
| Issue | Potential Cause | Resolution |
| :--- | :--- | :--- |
| Price data not loading | Supabase API limit | Check Supabase dashboard and refresh connection string. |
| UI misalignment | Browser cache | Clear browser cache or use Incognito mode. |

## Escalation Policy
1. **Level 1 (General):** Initial response within 24 hours.
2. **Level 2 (Technical):** Escalated to Developers if logic fix is required.
3. **Level 3 (Infrastructure):** Escalated to DevOps for hosting/database failures.

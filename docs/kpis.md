# Key Performance Indicators (KPIs)

1. **Deployment Frequency**: How often code is successfully merged to main and deployed.
2. **First Contentful Paint (FCP)**: The time it takes for the browser to render the first piece of DOM content (UX Metric).
3. **Build Success Rate**: Percentage of GitHub Actions/Vercel builds that pass without errors.
4. **Lead Time for Changes**: The amount of time it takes for a commit to go from 'dev' to 'production'.
5. **Static Site Availability (Uptime)**: The percentage of time the site is accessible to users.

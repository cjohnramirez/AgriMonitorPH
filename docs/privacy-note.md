# Privacy Note

## Data Collection
As a static demonstration site, AgriMonitorPH **does not** collect, store, or process personal identifiable information (PII) in a backend database.

## Usage Data
* **Cookies:** We use standard local storage for session simulation (e.g., mock auth tokens).
* **Retention:** Local storage data remains on the user's device until cleared.
* **User Rights:** Users have the right to browse anonymously. Since no data is stored on our servers, no "Right to Erasure" request is necessary.
